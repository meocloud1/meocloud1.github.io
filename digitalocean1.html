#Intro


Install Windows on Digital Ocean


In this section, we will learn how to Install Windows on DigitalOcean. There are two methods:


  • 1. Custom Images

  • 2. Recovery Method

#Using Custom Images


Video Tutorial


- Step One


  1. Go to Manage - Image - Custom image - Import via URL

  2. - Step Two

  3. Choose any Windows GZ file for your Droplet OS with Import via URL; it will take about 1 hour.
  4. After the Windows OS is successfully uploaded, go to Droplets - Create - Droplets


  5. - Step Three

  6. Choose any Region you like.

  7. On Choose an image, choose custom image, and select the Windows OS you just uploaded.

  8. Choose Droplets type and create.

  9. - Step Four

  10. Open it with MSTSC or any Application for RDP.

    Summary


    Cards

    Advantages

    Save time on the next Windows installation; no need to install every time you make a new droplet.

  • - Disadvantages: Takes 1 hour or more to upload new images.
