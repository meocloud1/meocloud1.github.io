Using Custom Image

Using Custom image for Windows installation

  1. Go to Manage - Image - Custom image - Import via URL

  2. Choose any Windows GZ file for your Droplet OS with Import via URL; it will take about 1 hour.
  3. After the Windows OS is successfully uploaded, go to Droplets - Create - Droplets
  4. Choose any Region you like.
  5. On Choose an image, choose custom image, and select the Windows OS you just uploaded.
  6. Choose Droplets type and create.
  7. Open it with MSTSC or any Application for RDP.
  • Advantages: Save time on the next Windows installation; no need to install every time you make a new droplet.
  • Disadvantages: Takes 1 hour or more to upload new images.
