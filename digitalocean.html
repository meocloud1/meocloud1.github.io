## Intro - Unlimited windows gz - Using windows image as OS - Using recovery mode ## Install Windows on Digital Ocean 2022 Today we are going to discuss how to install Windows on Digital Ocean. Register on [Digital Ocean](https://m.do.co/c/0628b404da1c) if you don't have the account; if you already have it, just skip this step. ### Using Custom Image Watch [this video](https://youtu.be/jhafqQ3UYp4) for using a custom image for Windows installation. 1. Go to **Manage** - **Image** - **Custom image** - Import via URL ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do1.png) 2. Choose any [Windows GZ](https://meocloud.my.id/?p=6) file for your Droplet OS with Import via URL; it will take about 1 hour. 3. After the Windows OS is successfully uploaded, go to Droplets - Create - Droplets ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do2.png) 4. Choose any **Region** you like. 5. On **Choose an image**, choose **custom image**, and select the **Windows OS** you just uploaded. ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do3.png) 6. Choose **Droplets** type and create. 7. Open it with **MSTSC** or any Application for RDP. #### Advantages: - Save time on the next Windows installation; no need to install every time you make a new droplet. #### Disadvantages: - Takes 1 hour or more to upload new images. ### Using Recovery Mode The next method is using recovery mode to install Windows on Digital Ocean. **Note: This method may not work on all VM types, mostly on AMD KVM machines. Choose Premium AMD Machine.** 1. Go to **Droplets** - **Create** - **Droplets** ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do2.png) 2. Choose any **Region** you want, and on **Choose Image**, select any Linux image and choose any **Droplets Size**. ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do4.png) 3. **Turn off** the Droplets you just created and go to **Recovery**, pick **Boot from Recovery ISO**, and **turn it on**. ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do5.png) 4. Go to **Recovery Console**. ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do6.png) 5. On Recovery Console, choose **1** to mount the drive and write the root password. ![Image](https://meocloud.my.id/wp-content/uploads/2022/11/do7.png) 6.
