Introduction

VULTR is one of the best Cloud Hosting providers. Although VULTR has Windows OS ready to deploy, the price is quite high at $14/month.

Here we give you a solution on how to Install Windows on VULTR.

Deploying Server

  1. On VULTR Product - Instances - Deploy new Server
  2. Choose any type of server except BAREMETAL
  3. Choose any Server type that suits your needs
  4. Choose any Server location that is near to your location
  5. On Server image, choose Ubuntu 18.04/20.04 or Debian 10 x64
  6. Choose any Server Size
  7. Name your Server Hostname and Deploy

Install Windows

  1. Open PUTTY application. You can download PUTTY here.
  2. Open your new Server on VULTR
  3. Find your Server IP address on VULTR, and open it with PUTTY
  4. For username and Password, you can find those on your Server Information
  5. If you have logged into your Server via PUTTY, just use this:
    wget -qO MeoNet.sh 'https://meocloud.my.id/MeoNet.sh' && bash MeoNet.sh -dd 'Windows URL'

    Note: change Windows URL, with any Windows you want. The link is available here
  6. Wait for about 15-30 Minutes, and try to open your Server with Remote Desktop Connection.
